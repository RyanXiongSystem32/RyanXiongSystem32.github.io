平均分

一年级1班有i个人，他们刚刚举行了一次数学考试，分数有你输入，输入i个人的分数（一次输一个），输出平均分。

请你用Python和C++实现一下。

解答：此重复类型需要用到for循环函数。

for循环在Python中的用法：
for i in range(重复次数):
重复循环内容代码

for循环在C++中的用法：
for(变量=1, 变量<变量++){
}
其中“变量=1”是变量初始化，“变量<”是for循环运行条件，“变量++”是变量的变化过程。

区间是变量<=i，把这些增加的数值添加进sum变量中。最后，记得要除以i，计算平均数哟！

代码

Python源码：
i = input("i")
i = int(i)
sum = 0
b = 0
for a in range(i):
b = input("b")
b = int(b)
sum += b
sum = sum / i
print(sum)

C++源码：
#include <bits/stdc++.h>
using namespace std;
int main() {
int i;
scanf("%d", &i);
int sum;
sum = 0;
int b;
int a;
for (a = 1; a <= i; a++) {
scanf("%d", &b);
sum = sum + b;
}
sum = sum / i;
printf("%d", sum);
return 0;
}
